MyList (top)
Description: This page is a jump page to all the Anime Pages, on this page you can use filters to filter MyList and see some simple information on Animes.
   
Notes: About filters:
Filters are somewhat of a problematic issue, they work, not at 100%, some combinations may render the anime list blank, if that happens just click one of the show "All" filters and everything should be back to normal.

Filter Conjuctions:
Filter by Alphabeth letter and then by these filters works, Filter by Genre, Filter by Type, Filter by State.
For the Show/Hide filter to work with other filters, the filter needs to be activated first, the other way arround does not work.
The Has Awards filters can't be used to filter an already filtered By State list.

Author note: if anyboy can sort out the problems with filters, please do so and mail me with the corrected code.

Filter "show/hide": Use this filter to show animes by status:
   complete / incomplete (toggles between showing complete and incomplete animes),

Preferences (top)
Description: This page is used for you to set display preferences for Anime Pages.
 
Option "Expand Information onLoad": Use this option to enable or disable by default if on anime pages the Full Anime Information Table (title, genres, type, awards, etc) is shown, if you select yes, it is shown, if you select no, only the Simple Anime Information Table (title, eps, seen, rating and reviews) is shown.
Option "Expand Episodes onLoad": Use this option to enable or disable by default if on anime pages the File Information Tables (fID, group, size, qual, stat, state, storage, source and ed2k) is shown by default or no.
Option "Number of Animes per Page": Number of animes to be shown per page on MyList. Select from the list the value.
Option "Show Trade Page": Shows the trade function and all associated things.
Option "Default Title Display": Shows by default the selected title, Japanese Romanji is allways the default if english or kanji titles not found.
   
Search (top)
Description: Page used to search Animes on MyList, there are some advanced filter options available.
 
Usage: Simply select the filters you want to use and take care to follow the notes on the filter fields if there are any, then just press "Start Search" to start a search, animes matching the selected options will be displayed on the results section.
   
Trade (top)
Description: If the Trade function is properly configurated, this page serves the porpuse of sending a mail to the owner of the list with a trade request.
 
Usage: Simply select the animes you want in MyList, use a Animes Per Page selection in which you can see all the animes without switching to other pages, the trade function only sees saves the current page selected animes.
When you have made your selections, press "Open Trade Page" and fill in the input boxes, when you think all is ok press Send Mail, a confirmation box with a preview of the mail will then appear, confirm and send the mail.
